## 📦 Installation Instructions
1. Clone the repository:
    ```bash
    git clone https://github.com/Eul45/eyuelengida.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the local server:
    ```bash
    npm start
    ```

This will run the project locally on your machine.
